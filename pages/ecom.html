Ecommerce Application

Completed an e-commerce UI project demonstrating conditional prototyping in Figma as a learning experience.

Project Overview

For this case study, I undertook a project to create a more user-friendly and intuitive interface for an e-commerce app. The primary goal was to learn and demonstrate the application of conditional prototyping techniques using Figma to enhance user experience.

Challenge

The existing e-commerce app had a complex checkout process, leading to a high cart abandonment rate. The challenge was to streamline the checkout process and improve overall usability.

Design Process

User Research : Conducted user research to understand pain points and preferences of potential shoppers. Identified the key issues, including a confusing checkout flow.

Conceptualization : Created wireframes and low-fidelity prototypes to ideate and visualize potential solutions for the checkout process.

Conditional Prototyping : Implemented conditional prototyping in Figma to simulate user interactions based on specific scenarios. This involved creating conditional pathways for various user decisions during checkout.

Iterative Design : Refined the UI based on feedback and usability testing, making sure the conditional prototypes responded appropriately to user choices.
